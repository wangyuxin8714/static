@font-face 自定义字体

1、自定义服务器端字体   即将一些特殊的字体让浏览者从服务器调用。三步完成！（不用切图显示特殊字体了）
1-0、先将应用的字体复制到站点文件夹fonts中。
1-1、定义	@font-face{
		font-family:"fzzt";    必须
		src: url(fonts/fzxingshu7000.ttf) format(truetype);    必须
		src:local(hwcy.ttf); 定义客户端字体
	//以上两个属性必须设置	
		}
1-2、调用
	div{
		font-family:fzzt;
		font-weight: bold;
		font-variant：small-caps;
		}
名称任意
src指定字体文件包含在服务器上的某个地方  此属性只能在@font-face规则里使用
url用于设置服务器端字体文件路径
local指定浏览者系统内的本地字体
format 定义字体格式名称

@font-face  他主要是把自己定义的Web字体嵌入到你的网页中。
@font-face的语法规则：
Font-family：:此值指的就是你自定义的字体名称，最好是使用下载的默认字体
Src：此值指的是你自定义的字体的存放路径，可以是相对路径也可以是绝路径；


1-3、字体格式：
1-3、一、TureTpe(.ttf)格式：
.ttf字体是Windows和Mac的最常见的字体，是一种RAW格式，主流浏览器支持如：ie9  firefox3.5  chrome4  Safari3.1

1-3、二、OpenType(.otf)格式：
.otf字体被认为是一种原始的字体格式，其内置在TureType的基础上，
Safari3.1   firefox3.5  chrome4

1-3、三、Web Open Font Format(.woff)格式：
.woff字体是Web字体中最佳格式，他是一个开放的TrueType/OpenType的压缩版本，IE9+,Firefox3.5+,Chrome6+,Safari3.6

1-3、四Embedded Open Type(.eot)格式：
.eot字体是IE专用字体，可以从TrueType创建此格式字体,支持这种字体的浏览器有【IE4+】；

1-3、五、SVG(.svg)格式：
.svg字体是基于SVG字体渲染的一种格式,支持这种字体的浏览器有Chrome4+,Safari3.1


注意：各个浏览器对字体格式的支持是最大的区别：
    webkit/safari：支持TrueType/OpenType(.ttf)，OpenType PS(.otf)，iOS4.2+支持.ttf，iOS 4.2以下只支持SVG字体；
    Chrome：除webkit支持的以外，从Chrome 6开始，开始支持woff格式；
    Firefox：支持.ttf和.otf，从Firefox 3.6开始支持woff格式；
    Opera：支持.ttf、.otf、.svg。尚不支持woff Opera 11开始支持WOFF；
    IE：只支持eot格式，IE9开始支持woff

注意：目前，各个移动端系统的默认字体
ios 系统默认中文字体是Heiti SC默认英文字体是Helvetica默认数字字体是HelveticaNeue无微软雅黑android 系统默认中文字体是Droidsansfallback默认英文和数字字体是Droid Sans无微软雅黑
winphone 系统默认中文字体是Dengxian(方正等线体)默认英文和数字字体是Segoe无微软雅黑

特殊字体格式的兼容
@font-face {
	font-family: 'diyfont';
	src: url('diyfont.eot'); /* IE9+ */
	src: url('diyfont.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
		 url('diyfont.woff') format('woff'), /* chrome、firefox */
		 url('diyfont.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
		 url('diyfont.svg#fontname') format('svg'); /* iOS 4.1- */
}





2、图标字体的应用：将图标转为字体，用定义字体的方式来显示图标。三步完成。
	1-复制图标字体文件夹到站点内
	2-页面链接此文件夹内的iconfont.css文件
	3-应用处创建空元素如i，并class命名依调用图标样式如class="iconfont icon-tianmao"
		.iconfont已定义好的样式，固定。
		.icon-tianmao 具体的图标字体名称，到展示页上查找
	其图标颜色大小等属性按字体的属性定义！




2-2、iconfont三种应用
    	1、unicode（最原始、兼容性最好Ie6+、不支持多色。支持单色、渐变色）
		<i class="iconfont">&#66e;</i>
	2、font-class（unicode方式的变种，书写更直观，主要是解决unicode书写不直观，语意不明确的问题、兼容性良好Ie8+，不支持多色。支持单色）-推荐 
		<i class="iconfont icon-tianmao"></i>
	3、symbol（兼容性较差，支持 ie9+及现代浏览器、支持多色图标，svg渲染）未来的主流
		
2-3、渐变文字图标 -webkit-核心
	background:linear-gradient(0deg,#f00, #0f0);
    	-webkit-background-clip:text;
    	-webkit-text-fill-color:transparent;（文字填充色）


拓展：文字描边（-webkit-text-stroke：2px #f00；
-webkit-text-stroke-width: 2px;
  -webkit-text-stroke-color: #f00;）

3、多行省略：-webkit核心 移动端
	overflow : hidden;
	text-overflow: ellipsis;
	display: -webkit-box;
	-webkit-line-clamp: 2;
	-webkit-box-orient: vertical;
